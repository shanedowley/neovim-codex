# Neovim-AIDE Documentation

Welcome to the documentation for **Neovim-AIDE**.

Neovim-AIDE is an **AI-Assisted Software Engineering Environment for Neovim** built around **correctness, control, traceability and human-controlled AI assistance**.

The documentation is organised around two primary audiences:

- **Users** — engineers using Neovim-AIDE in their daily software engineering workflow.
- **Contributors** — engineers helping to design, build, validate and improve Neovim-AIDE.

By separating these audiences, the documentation remains focused, maintainable and easy to navigate as the project evolves.

# Project Principles

Neovim-AIDE is built around a small number of long-term engineering principles.

These principles guide both technical decisions and contributor workflows.

- Human-controlled AI assistance
- Correctness through explicit operations
- Observable system state
- Deterministic behaviour
- Sandbox-First Validation
- Reproducible engineering

Every change to the project should reinforce these principles.
